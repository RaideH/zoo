# PetCare Pro - Premium Pet Management System

PetCare Pro is a professional dashboard for pet owners to track health, routines, and milestones of their beloved companions. Built with React and a focus on visual excellence.

## ✨ Features
- **Dashboard**: Quick overview of your pets and upcoming tasks.
- **Health Tracker**: Maintain medical history and vaccination records.
- **Daily Routine**: Interactive scheduling for feeding, walking, and play.
- **Premium UI**: Modern, glassmorphism-inspired design with smooth animations.
- **SPA Navigation**: Fast, fluid transitions between states.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- pnpm / npm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```

## 🏗 Project Structure
- `src/components`: Reusable UI components (Buttons, Cards, Modals).
- `src/context`: Global state management for Auth and Pet data.
- `src/pages`: Application views (Landing, Auth, Dashboard, Details).
- `src/index.css`: Global design system and theme tokens.
